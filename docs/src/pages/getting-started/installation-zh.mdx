## 安装 & 使用

<br />

#### 安装

*****

用 npm 或者 yarn，将 `rc-multi-level-selector` 作为依赖安装：

```bash
# Yarn
$ yarn add rc-multi-level-selector

# NPM
$ npm install --save rc-multi-level-selector
```

> **注意：**
>
> `rc-multi-level-selector` 使用了 [getDerivedStateFromProps](https://reactjs.org/docs/react-component.html#static-getderivedstatefromprops) API，
> 请确保安装了大于或等于 v16.4.0 版本的 `react` 及 `react-dom`

<br />

#### 使用

*****

import MultiLevelSelector from '../../../../lib'
const options = [
  {
    id: 1, value: 'item 1',
    items: [ { id: 2, value: 'item 2' }, { id: 3, value: 'item 3' } ]
  },
  {
    id: 4, value: 'item 4',
    items: [ { id: 5, value: 'item 5' }, { id: 6, value: 'item 6' } ]
  }
]

简单的几行代码就可以完成一个级联选择器

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import MultiLevelSelector from 'rc-multi-level-selector'

const options = [
  {
    id: 1, value: 'item 1',
    items: [ { id: 2, value: 'item 2' }, { id: 3, value: 'item 3' } ]
  },
  {
    id: 4, value: 'item 4',
    items: [ { id: 5, value: 'item 5' }, { id: 6, value: 'item 6' } ]
  }
]

ReactDOM.render(
  <MultiLevelSelector
    options={options}
    onChange={console.log}
  />,
  document.querySelector('#app')
);
```

**效果：**

<MultiLevelSelector
  options={options}
  onChange={console.log}
/>
